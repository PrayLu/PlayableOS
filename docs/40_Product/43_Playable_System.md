# Playable System

> Playable 是 PlayableOS 的核心产品能力。

它负责将企业知识转化为可体验、可互动、可成长的训练内容。

---

# 1. Definition

Playable 是 AI 自动生成的能力训练体验。

每一个 Playable 对应一个明确的训练目标。

例如：

- 新员工入职
- 销售异议处理
- 客户投诉处理
- 安全生产
- 企业文化

---

# 2. Playable Flow

```text
企业知识

↓

AI 生成 Playable

↓

员工体验

↓

AI 反馈

↓

能力成长
```

---

# 3. Playable Types

V1 支持：

- AI 对话
- 情景模拟
- 决策挑战
- 流程演练
- 角色扮演
- Boss Challenge

---

# 4. Playable Features

每一个 Playable 包括：

- Training Goal
- Scenario
- Interaction
- Feedback
- Result

---

# Summary

Playable 是连接企业知识与员工能力的桥梁。
