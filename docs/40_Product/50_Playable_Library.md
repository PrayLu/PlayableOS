# Playable Library

> Playable Library 定义 PlayableOS 中所有 Playable 的统一设计标准。

---

# 1. Definition

每一个 Playable 都遵循统一结构。

Playable 不是小游戏。

Playable 是一个能力成长单元。

---

# 2. Standard Structure

```text
Playable

├── Goal
├── Scenario
├── Roles
├── Interaction
├── Challenge
├── Feedback
├── Assessment
└── Reward
```

---

# 3. Categories

V1 包括：

- Dialogue
- Scenario
- Workflow
- Decision
- Role Play
- Boss Challenge
- Simulation

---

# 4. Metadata

每个 Playable 包括：

- Name
- Description
- Capability
- Duration
- Difficulty
- Target Role
- Industry
- Tags

---

# 5. Design Rules

每个 Playable：

只有一个训练目标。

每次体验：

5~15 分钟。

必须：

有反馈。

有成长。

有结果。

---

# Summary

Playable Library 是 PlayableOS 的能力组件库。

AI 所有生成内容。

均遵循统一标准。
