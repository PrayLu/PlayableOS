# Assessment System

> Assessment 用于持续评估员工能力成长。

---

# Assessment Sources

AI 综合分析：

- 行为
- 决策
- 对话
- 完成情况
- 重试记录

---

# Assessment Output

输出：

- 能力评分
- 成长建议
- 优势分析
- 待提升能力

---

# Enterprise Dashboard

企业可查看：

- 团队能力分布
- 部门成长趋势
- 高潜人才
- 共性问题

---

# Summary

Assessment 不评价学习。

Assessment 评价能力成长。
