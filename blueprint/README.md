## 查看网页版

构建并本地预览：

```bash
cd blueprint
npm install
npm run build
npm run serve
```

浏览器打开 [http://localhost:3456](http://localhost:3456)

或直接双击打开 `blueprint/index.html`。

修改 `.md` 章节后重新运行 `npm run build` 即可更新网页。
